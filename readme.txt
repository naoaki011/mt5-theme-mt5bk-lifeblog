★当テーマの使い方

1.ファイルのアップロード
　解凍してできたフォルダのうち、mt-static／plugins／themesの3つのフォルダを、Movable Type 5のインストール先ディレクトリにアップロードします。

2.ウェブサイトとブログの作成
　書籍57ページの「テーマのインストール」の手順に従って、ウェブサイトとブログを作成します。

3.カスタムフィールドの設定
　書籍69ページの「カスタムフィールド」の手順に従って、プロフィール画像およびプロフィール情報の入力を行ないます。

4.アクションストリームの設定
　書籍79ページの「アクションストリーム」の手順に従って、利用サービスの追加とCRONジョブの設定を行ないます。

5.カテゴリーの設定とブログ記事のインポートおよびSEOに対応するテンプレートの書き換え
　書籍91ページの「基本機能で行うSEOカスタマイズ」の手順に従って、カテゴリーの設定とブログ記事のインポート、SEOに対応するテンプレートの書き換えを行ないます。

6.CKEditor for Movable Typeの設定
　書籍111ページの「WYSIWYGエディタ」の手順に従って、CKEditor for Movable Typeの設定を行ないます。

7.カテゴリの並べ替え
　書籍134ページの「カテゴリの並べ替え」の手順に従って、カテゴリの並べ替えを行ないます。

★当テーマでの注意点

1.このテーマは個人で運営するブログである事を前提に、作成されています。ユーザー情報の取得に<mt:Authors need_entry="1">を使っているため、複数人で運営するケースでは同じ内容が重複して出力されるなどの問題が発生する場合があります。

2.このテーマではウェブサイトの下に個人ブログが一つというケースを想定して作られています。ウェブサイト配下のブログを複数にした場合には、ナビゲーションメニューでの表示など、幾つかの箇所で期待する表示にならない可能性があります。

3.このテーマ用に同梱されているアクションストリームプラグインは、藤本壱さんが配布している アクションストリームプラグイン（MT5用）る http://www.h-fj.com/blog/archives/2009/11/15-171130.php に対して、幾つかのアクションを追加してあります。テーマでインストールされる幾つかのモジュール（Amebaなう・GoogleBuzz・ニコニコ動画・ブクログ）で、追加したアクションを表示可能になります。
なおこの追加分のアクションについては、小野崎直昭が作成した NicheActionsプラグイン http://www.zelazny.mydns.jp/pub/plugins/niche-actions.php が元になっています。追加アクションに対する質問・要望は藤本さん宛ではなく、小野崎の方へお出しくださるようお願いいたします。

4.被はてブウィジェットを使用する場合には、藤本壱さんが配布している はてな被ブックマーク情報表示アクションストリームプラグイン http://www.h-fj.com/blog/archives/2009/03/30-180713.php を追加する必要があります。

5.アイテムウィジェットでのサムネイルを作成する時に、<$mt:AssetThumbnailURL square="1" width="76"$>というMTタグで76pixel四方の画像を生成し、 width="75" height="75"というように75pixelで表示をさせています。これはMovable Type 5.01までで再現する「mt:AssetThumbnailxxxタグで square="1"を指定した時に、widthまたはheightとして75を指定すると、squareオプションが無効になる」というバグに対応する為です。このバグは時期リリースでは解決される予定です http://bugs.movabletype.org/default.asp?102463。バグが解決されたならば、<$mt:AssetThumbnailURL square="1" width="75"$>の指定に書き換えることをお勧めします。

6.あまり話題になっていないようですが、幾つかのサーバーでdeliciousからのフィード取得が制限されているようです。http://feeds.delicious.com/rss/dakinyの更新に失敗しました: 999 Unable to process request at this time -- error 999 の様なエラーログが残っている場合、それはdelicious側の問題と考えて構わないと思います。
